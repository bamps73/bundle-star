# bundle-star

bundle-star starred gem's github repository when `bundle-star install`ed

## Installation

Install it yourself as:

    $ gem install bundle-star

## Usage

Use `bundle-star` instead of `bundle`:

    $ bundle-star install
    $ bundle-star update

Pro tip:

    alias bsi="bundle-star install"
